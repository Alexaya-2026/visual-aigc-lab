# 审美阶梯路线图
### 从"看多了"到"看得出为什么" —— 五层 + 拉片仪式

这份文档补的是 `README.md` 3.5 节 (b) 标出的缺口:审美阶梯此前只有一张五层表格,没有具体读什么、怎么练、什么时候算过关。这里补齐,格式对照 `comfyui-mastery-roadmap.md`。

## 大局观(先看这张表)

| 层 | 主题 | 核心读物 | 每日 rep | 出师标志 |
|---|---|---|---|---|
| ① | 构图基础 | 《Picture This》 | 每天挑 1 张图,写清楚"哪里舒服/哪里别扭,为什么" | 拿到一张没见过的图,能不靠参考主动判断构图好坏 |
| ② | 视觉叙事 | 《Framed Ink》+《The Visual Story》 | 分镜/画面叙事拆解 | 一组画面不靠文字说明,自己能讲出故事 |
| ③ | 色彩与光影 **★建议插队,优先级高于①②的读完顺序** | 《Light: Science & Magic》 | 每天记录 1 个打光案例,尤其金属/玻璃/液体 | 能主动设计光而非碰运气,**且能把结论迁移进电商实拍** |
| ④ | 镜头语言 | 《The Five C's of Cinematography》 | 拉片笔记:景别/运镜/节奏 | 懂景别运镜节奏,视频不"散"(**实操场地要等技术阶梯④激活,现在先囤知识**) |
| ⑤ | 风格与个人语言 | 综合以上 + 个人 mood board | 维护 mood board,持续筛选 | 作品一看就是"你的",不用解释 |

**为什么③要插队**:RESOURCES.md 里这本书自己写得很直白——"专攻金属/玻璃/液体等难物体(正中电商主业)"。这不是巧合式的加分项,是这本书存在的理由。①②按顺序读没问题,但③不要拖到最后才碰,它是唯一一层能直接回哺主业 80% 的。

**为什么④暂时只囤知识不深练**:镜头语言的出师标准里有"视频不散",但你现在没有视频实操场地——技术阶梯④还没激活(见 README 3.5 节)。这层现在的任务只是"拉片记笔记攒理解",不需要产出成片来验证,等④激活后再回来补"能不能真的做出不散的视频"这道题。

## 怎么用这张表(方法论)

审美阶梯和技术阶梯的练习心法不一样,别把技术阶梯"改一个变量"的纪律原样套过来:

1. **一张图只拆一个原理,别想着榨干它。** 一张参考图里同时有构图、色彩、光影、叙事四件事在发生,一次全解构,你只会记住"这张图很牛",记不住任何可迁移的判断力。today's rep 挑一个角度,写透就收。
2. **"为什么"必须落成文字,不能停在感觉里。** "这张图很舒服"不是审美阶梯的产出,"这张图舒服是因为视觉重心落在三分线交点、留白给了呼吸感"才是。写不出来,说明还没真的看懂,只是眼熟。
3. **拉片是日常仪式,不是任务清单。** 每天 10–15 分钟,雷打不动,比一周攒一次两小时的效果好得多——这和技术阶梯 roadmap 里"高频短练习 > 低频长练习"的建议是同一个道理。
4. **纸质书按你自己的节奏读,不设周数。** 技术阶梯②③④按周估算是因为工具会过期、需要抓紧;审美阶梯这几本书耐用不过时,读快读慢不影响正确性,别为了"进度"牺牲理解深度。
5. **每层至少每周产出一次深度拆解,存进 `aesthetics/`。** 日常拉片可以很轻,但每周挑其中一次,认真拆解写清楚,这是防止"每天扫一眼但什么都没沉淀"的护栏。

## 拉片笔记模板(可复用,存进 `aesthetics/`)

```
日期:
来源:(ArtStation / Behance / Film-Grab / ShotDeck / 其他)
对应层:①构图 / ②叙事 / ③色彩光影 / ④镜头 / ⑤风格
这次只拆一个原理:
好在哪(具体到可迁移的判断,不是"好看"):
如果换个方向会怎样(反事实,逼自己想清楚机制):
能不能用在主业电商图里:是/否 + 一句话
```

最后一格是这份文档和主业之间的钩子——不用每次都填"是",但每次都问一遍,你会慢慢发现哪些原理是真的高频有用的。

## 每层出师自测(不用跑代码,自己回答)

- **①** 给你一张没见过的图,你能不能在 1 分钟内说出它构图上的一个明确优点或缺点,而不是"感觉还行"?
- **②** 一组 3–5 张的画面,拿掉所有文字说明,别人还能看出你想讲的故事吗?
- **③** 你能不能主动为一个金属或玻璃产品设计一个打光方案,并且提前预判会有什么反光/阴影问题,而不是拍完再补救?
- **④** 看一段陌生视频,你能不能说清楚它每个镜头的景别和切换节奏,以及为什么这么剪感觉不"散"?
- **⑤** 陌生人看你近期的作品,能不能不看署名就认出是你的风格?

五个都能给出具体理由(不是"能"或"不能"这种是非题),这层才算真出师。

## 常见坑

- **把拉片当成刷图。** 刷了很多但没写"为什么",等于没练——这是审美阶梯最容易假装在进步的地方。
- **等④激活了才想起来读书。** ④现在就可以读,只是不强求产出成片验证,别把"读"和"练"混成同一件事拖延。
- **为了不落后而牺牲③的深度。** ③是唯一直接反哺主业的层,值得比①②多花时间,不必因为编号在前面就先紧后松。
