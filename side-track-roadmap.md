# 副业·从零创作路线图
### 从"素材库攒着"到"真的在发东西"

补的是 `README.md` 3.5 节 (b) 的另一半缺口:副业 20% 此前只有一句"角色/场景/世界观/短视频/文案",没有具体怎么练、用什么工具、什么时候算真的在做。这里补齐。

---

## 先讲清楚一个容易搞反的前提

主业的核心心法是"托管当参照组,本地练真功夫"——因为主业目标是**控制力**,这是全景图文档里反复强调的。

**副业不是这样。副业目标是广度探索和真实产出,不是练控制力。** 所以副业应该反过来用:**直接用托管成品(MJ / Krea / Leonardo / Kling / Runway / Veo)出活,不需要等 ComfyUI 本地能力练到位。**

如果把主业那套"必须练到本地护城河"的纪律搬到副业上,会导致一个具体的坏结果:技术阶梯④(视频)还没激活,副业里的短视频探索就被无限期押后——但④没激活跟副业能不能做短视频,其实没有必然关系,托管视频工具现在就能用。

## 大局观

| 阶段 | 是什么 | 用什么工具 | 出师标志 |
|---|---|---|---|
| A. 被动积累 | 主业 report-back 顺手喂进博客素材库(**已经在跑**,见 README 核心循环) | 不需要额外工具 | 素材库攒到能看出重复出现的话题/风格苗头(比如 10+ 条) |
| B. 从零创作练习 | 角色/场景/世界观的独立创作练习 | MJ V8.1 / Krea / Leonardo(图);Civitai 抄作业找方向 | 能独立想一个概念,用托管工具几次内出到满意结果,不靠现抄别人的完整提示词 |
| C. 短视频初探 | 把 B 阶段的角色/场景接上动态 | Kling / Runway / Veo(托管,**不等本地 Wan/LTX**) | 能做出一个 30 秒以内、有基本叙事节奏的片段,即便全靠托管工具拼出来 |
| D. 发布节奏 | 把素材库和 B/C 阶段产出真正整理发出去 | 公众号 / B站 / 知乎(RESOURCES.md 里的中文阵地) | 建立一个可持续的发布 cadence,不是一次性爆发然后停摆 |

A、B、C 可以并行推进,不是严格顺序——A 本来就在背景里自动跑,B、C 看你哪天有空当天想做哪个。**D 是唯一需要单独安排时间的**,因为"发布"这个动作不会像 A 一样自动发生。

## 方法论(和主业不一样,别混用)

1. **广度优先,数量优先于精度。** 主业改一个变量、追求可控可复现;副业刚好相反——先做够量,再回头看哪些方向值得深挖。太早对副业上单变量纪律,会把探索速度拖到主业的节奏,浪费了副业本该有的自由度。
2. **但评估纪律不能丢。** 每次创作还是要写一句"为什么选这个构图/风格/概念",哪怕只有一句话——不然副业会退化成"随便点几下,凑数发出去",学不到任何东西,也攒不出真正的个人风格。
3. **Civitai 是起点不是终点。** 抄别人的提示词找方向没问题,但 B 阶段的出师标准明确要求"不靠现抄",这是逼自己从"复现别人的效果"过渡到"自己想要什么效果"。
4. **C 阶段做不出效果,不代表你缺技术。** 托管视频工具本身的控制力有限,做不出想要的东西时,先问"是不是工具的天花板到了",而不是急着去学本地视频管线——那是技术阶梯④的事,触发条件见 README 3.5 节,不要为了副业的一次卡壳就提前打乱主业节奏。

---

## 什么时候值得把副业迁回本地/护城河工具

这是一个具体的、可判定的触发条件,不是"感觉需要更多控制力"这种模糊冲动:

> 你在托管工具上反复想要同一种效果,调了很多次都调不出来,并且这个效果不是一次性的、是你会持续需要的。

满足这条,才是把某个副业需求正式带回技术阶梯的合理理由——呼应主 roadmap 护栏里"不为新换工具,撞到真问题才引入"的心法,副业这边同样适用,只是触发方向反过来:不是"看到新工具就想用",而是"托管工具明确到顶了才考虑本地"。

## 常见坑

- **把副业当成主业的技术练兵场。** 不需要等本地视频能力成熟才敢做短视频——见前面"先讲清楚一个容易搞反的前提"。
- **A 阶段一直空转,从不进 D。** 素材库攒了很多条从来不整理发布,等于副业只做了一半——A 是燃料,D 才是产出。
- **B 阶段一直在抄,从不断奶。** Civitai 找方向没问题,但如果每次都是完整照抄别人的提示词,那不算"独立创作练习",出师标准里"不靠现抄"这条要真的守住。
- **C 阶段卡壳就想着自己练视频模型。** 先确认是不是真的撞到了"反复需要、调不出来"这条硬指标,不是随手卡一次就升级成大工程。
